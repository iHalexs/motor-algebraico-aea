# 🧮 AEA - Motor Algebraico 

Calculadora avanzada de Álgebra Lineal desarrollada en Python. Proyecto final diseñado para resolver y explicar paso a paso operaciones matriciales complejas.

## 🚀 Características Principales
* **Espacios Nulos:** Calcula la matriz escalonada reducida y extrae la base del Nul A.
* **Conjuntos de Bases:** Identifica las columnas pivote para la base del Espacio Columna.
* **Sistemas de Coordenadas:** Encuentra el vector de coordenadas respecto a una nueva base.
* **Diagonalización:** Calcula Valores Propios, Vectores Propios y genera las matrices P, D y P^-1.

## 🛠️ Tecnologías Usadas
* **Python 3**
* **Streamlit** (Interfaz gráfica y despliegue web)
* **SymPy** (Motor de cálculo simbólico)
* **Pandas** (Estructuración de datos)
